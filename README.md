Descripción breve del proyecto:
SOE-SautaYork es una plataforma digital desarrollada para mejorar los procesos de reclutamiento y vinculación laboral entre universidades, egresados y empresas. El sistema permitirá administrar vacantes, perfiles profesionales, evaluaciones y reportes interactivos de manera segura, organizada y eficiente mediante tecnologías web modernas.
