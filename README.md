Descripción breve del proyecto:
SOE-SautaYork es una plataforma digital desarrollada para mejorar los procesos de reclutamiento y vinculación laboral entre universidades, egresados y empresas. El sistema permitirá administrar vacantes, perfiles profesionales, evaluaciones y reportes interactivos de manera segura, organizada y eficiente mediante tecnologías web modernas.

Archivos de ejemplo incluidos:
- docs/examples/vacante-ejemplo.json
- docs/examples/perfil-ejemplo.json
- docs/examples/empresa-ejemplo.json

Simulación previa al push:
Se agregó un script de verificación en scripts/simular-push.ps1 y un hook de Git en .githooks/pre-push.cmd para ejecutar una comprobación antes de hacer push.

Para activarlo en Git, ejecuta:
- git config core.hooksPath .githooks

Y luego realiza un push para probar la simulación.
